# Maison Constantine

## Installation sur un nouvel ordinateur

1. **Transférer le dossier** : Copiez ce dossier sur votre nouvel ordinateur.
2. **Installer Node.js** : Assurez-vous d'avoir Node.js installé.
3. **Installer les dépendances** :
   Ouvrez un terminal dans le dossier et lancez :
   ```bash
   npm install
   ```
4. **Lancer le site** :
   ```bash
   npm run dev
   ```
